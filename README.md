# 委員書面審查系統

## 使用方式

正式委員連結格式：

https://any192204-hub.github.io/committee-written-review-system/?rid=R001&token=委員token&name=委員姓名

## 系統功能

- A / B / C 評分
- 審查意見填寫
- Google Sheets 即時同步
- 本機暫存與補送
- Word 評分表產出

## 注意事項

- 不要將正式委員 token 寫入 README
- Apps Script Web App 需部署為「任何人」可存取
- Google Sheets reviewers 工作表需設定 reviewerId、token、active
